# quarkzLib
Funktions- und Klassenbibliothek.

## units

* **core\qzLib.Core.DormConfigBuilder**:
Builderklasse zum Erzeugen des Inhaltes der Konfigurationsdatei für DORM (Delphi ORM) zur Laufzeit.

* **core\qzLib.Core.DormLoggers**:
Dummy-Logger für DORM.

* **core\qzLib.Core.HtmlViewerPdfWriter**:
PDF-Writer zum Exportieren des Inhaltes einer THTMLViewer-Komponente.
